💻 Tema do Sistema

A **TechStore** é um sistema de catálogo e gerenciamento de produtos de informática. O objetivo é oferecer uma plataforma moderna para visualização de produtos, cadastro e login de usuários, solicitação de orçamentos e gerenciamento de pedidos.

---

 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Supabase
- Git
- GitHub

---

# 📖 Explicação do Projeto

O projeto TechStore foi desenvolvido para simular uma loja virtual especializada em produtos de informática.

O sistema permite que o usuário:

- Visualize os produtos disponíveis;
- Crie uma conta;
- Faça login;
- Acesse sua conta;
- Solicite orçamentos;
- Visualize pedidos;
- Navegue pelas páginas Sobre Nós e Contato.

O sistema possui uma interface responsiva, moderna e intuitiva.

---

# 🗄️ Uso do Supabase

O Supabase foi utilizado como banco de dados do projeto.

Ele é responsável por armazenar informações como:

- Cadastro de usuários;
- Dados de login;
- Informações utilizadas pelo sistema.

A comunicação com o banco é feita através da biblioteca JavaScript oficial do Supabase utilizando sua API.
